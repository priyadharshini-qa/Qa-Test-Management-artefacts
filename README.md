# 🧪 Test Management artefacts

Welcome to the **Test Mangement artefacts** repository!
This repository contains various QA process/reports templates for QA leads.

## 📂 Repository Structure

```
software-testing/
├── Agile_Test_Plan_Template.docx
├── PSM_Templates_Reference_Kit.docx
├── QA_Daily_Status_Report.docx
├── Test_Closure_Report_template.docx
├── Test_Metrics_Template.xlsx
├── LICENSE
└── README.md
```
---

## 📄 Template Descriptions

| File | Description |
|------|-------------|
| `Agile_Test_Plan_Template.docx` | Test plan template aligned with Agile/Scrum methodology |
| `PSM_Templates_Reference_Kit.docx` | Reference kit with process and project management templates |
| `Test_Closure_Report_template.docx` | End-of-cycle test closure report template |
| `Test_Metrics_Template.xlsx` | Tracks execution metrics, defect counts, and sprint-wise quality summary |
| `QA_Daily_Status_Report.docx` | Daily QA status tracker with test execution progress and defect log |

## 🚀 How to Use

**Step 1:** Clone the repository
```bash
git clone https://github.com/priyadharshini-qa/Qa-Test-Management-artefacts.git
```

**Step 2:** Open the project folder
```bash
cd Qa-Test-Management-artefacts
```

**Step 3:** Open any template relevant to your testing phase and customise it for your project.
